Hexo搭建博客指南

发表于 2017-05-13

安装

  1. 安装Homebrew
  2. 安装Node:brew install node
  3. 安装hexo:sudo npm install -g hexo-cli

使用

  1. 初始化博客目录（以Blog为目录名）:
    hexo init Blog
  2. 参考在 hexo 中无痛使用本地图片进行以下配置:
    • 首先确认 _config.yml 中有 post_asset_folder:true。
    • 然后在hexo目录下(即Blog目录下)执行:
      npm install https://github.com/CodeFalling/hexo-asset-image --save
  3. 创建文章（以Article为文件名，后续可修改md文件中的title字段来修改文章名）:
    hexo n Article
    PS:完成第一步中的图片配置后，会在source/_posts中同时生成Article.md和 Article文件夹，把图片(img.png)放在Article文件夹下并在md文件中使用Article/img.png即可成功访问，并在生成静态页面的过程中成功转换成相对路径。
  4. 生成静态页面ss:
    hexo g
  5. 启动本地服务预览:
    hexo s
    PS:启动后访问 http://localhost:4000/

配置主题（以Next为例）

  1. 下载Next主题到themes/next目录下:

    git clone https://github.com/iissnan/hexo-theme-next themes/next

  2. 修改_config.yml中的theme字段为next

  3. 重新生成静态文件。

    PS:Next主题更多详细配置参见官方文档

部署到GitHub

  1. 在GitHub创建新的仓库，命名为\.github.io。(例如，我的GitHub账户名为IceCreamWu，则创建名为IceCreamWu.github.io的仓库);
  2. 配置_config.yml中的deloy字段:
    deploy:
    type: git
    repo: https://github.com/IceCreamWu/IceCreamWu.github.io.git
    branch: master
  3. hexo目录下执行：
    hexo g hexo d
    PS:如果发生ERROR Deployer not found: git,便先执行以下命令：
    npm install hexo-deployer-git --save
